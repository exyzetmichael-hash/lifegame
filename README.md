# LifeQuest

Игровой трекер жизни: таймер времени (секундомер / помодоро / обратный отсчёт), привычки, задачи, XP и статы персонажа, ачивки, дашборд по дню/неделе/месяцу. PWA — ставится на телефон, работает офлайн.

## Стек

React + TypeScript + Vite, Tailwind CSS v4, Zustand (локальное хранилище), Framer Motion, Recharts, react-router-dom. Supabase — опционально, для облачной синхронизации.

## Запуск

```bash
npm install
npm run dev
```

Приложение полностью работает локально (localStorage), без каких-либо ключей.

## Облачная синхронизация (опционально)

Создай проект на [supabase.com](https://supabase.com), скопируй `.env.example` в `.env` и заполни:

```
VITE_SUPABASE_URL=...
VITE_SUPABASE_ANON_KEY=...
```

## Сборка

```bash
npm run build
```
